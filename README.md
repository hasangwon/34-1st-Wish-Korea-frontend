# wish korea

## 위시 코리아

###### ㈜위시 코리아 (Wish Korea)는 친환경적이며 동물, 자연과 함께하기 위해 늘 차별 없이 윤리공정을 지키는 기업입니다. 우리는 화장품, 욕실 용품 상품을 판매하고 있으며, 이국적이고 개성적인 향을 가진 핸드메이드 상품을 판매하고 있습니다.

<b>팀원 FE) 정예지, 안중영, 우혜림, 하상원 / BE) 정진관 </b>
